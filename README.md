#🧱 Tiles Gallery



A responsive tile showcase web application built with Next.js, where users can browse beautiful tile collections, view details, and manage their profiles through a modern interface.



#🌐 Live URL


https://thetilesgallery.netlify.app/



#✨ Key Features

Responsive and modern UI

Featured Tiles and All Tiles pages

Dynamic tile data loaded from JSON

Email/Password and Google Authentication

User Profile page

Protected tile details for logged-in users

Custom loading and 404 pages
